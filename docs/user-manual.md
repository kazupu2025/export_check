# ユーザーマニュアル — 輸出管理 該否判定システム

> 対象: 輸出管理担当者  
> 最終更新: 2026-05-16

---

## このシステムでできること

外国為替及び外国貿易法（外為法）の貨物等省令に基づき、輸出予定品目が**輸出許可申請の対象かどうか**を自動判定します。

- 品目のスペック（実測値）を入力すると、法令の規制閾値と数値比較して判定
- 判定結果と根拠条文を画面に表示
- PDF出力して証跡として保管

**⚠️ 重要**: このシステムの判定は参考情報です。最終的な該否判断は担当者が確認し、不明な場合はCISTECへご相談ください。

## ログイン・パスワード管理

### ログイン

システムへのアクセスにはログインが必要です。

1. ブラウザでシステムのURLを開く
2. メールアドレスとパスワードを入力して **「ログイン」** をクリック
3. ホーム画面に移動します

ログアウトは左サイドバー下部の **「ログアウト」** から行えます。

### パスワードを忘れた場合

1. ログイン画面下部の **「パスワードを忘れた方はこちら」** をクリック
2. 登録済みのメールアドレスを入力して **「リセットメールを送信」** をクリック
3. 届いたメール内のリンクをクリック
4. 新しいパスワード（8文字以上）を入力して **「パスワードを更新」** をクリック

> **注意**: リセットメールのリンクには有効期限があります。届いたらすみやかに操作してください。

## 該否判定の手順

### Step 1: 品目入力

1. 左メニューの **「該否判定」** をクリック
2. 品目名を入力する（例: `ガラス繊維織物`、`マシニングセンタ`）
   - カンマ区切りで複数入力可
3. 品目の形態を選択する

| 形態 | 選ぶべき品目 |
|------|------------|
| 繊維・素材 | 炭素繊維、ガラス繊維、CFRPシートなど素材単体 |
| 成型品・構造材 | CFRP板、複合材料管など加工後の製品 |
| 工作機械・加工装置 | マシニングセンタ、旋盤、NC工作機械 |
| レーザー・光学機器 | レーザー発振器、光学センサー |
| 電子機器・半導体 | 集積回路、コンピュータ、通信機器 |
| 化学品・特殊材料 | 特殊金属、化学製剤 |
| 不明・その他 | どの形態にも当てはまらない場合 |

4. **「規制パラメータを検索」** ボタンをクリック

---

### Step 2: スペック入力

関連する規制パラメータが表示されます。各パラメータに実測値を入力してください。

- **閾値基準**（青字）が比較対象の規制値です
- 不明なパラメータは**空欄で構いません**（空欄のパラメータは判定対象外になります）
- 単位は画面右に表示されている単位に合わせてください

**パラメータの説明**

| パラメータ | 意味 |
|-----------|------|
| 含有量 | SiO₂など主成分の含有割合（%） |
| 比強度 | 引張強さ ÷ 密度（m） |
| 比弾性率 | 弾性率 ÷ 密度（m） |
| 軸数 | 繊維の積層方向数（1=UD、2=縦横、4=±45°含む） |
| 昇華温度 | 昇華開始温度（℃） |
| 位置決め精度 | 工作機械の位置決め誤差（mm） |
| 出力 | レーザーの出力（W） |

入力後、**「該否判定を実行」** をクリック。

---

### Step 3: 判定結果の確認

#### 判定結果の種類

| 結果 | 意味 | 対応 |
|------|------|------|
| 🔴 **許可必要** | 規制閾値を超えており、輸出許可申請が必要 | 経産省へ申請 |
| 🟢 **非該当（許可不要）** | 規制対象外。通常の輸出手続きで可 | 証跡を保管 |
| 🟡 **要人間確認** | 入力値が不足または関連閾値なし | CISTECへ確認 |

#### パラメータ比較表の見方

- **超過 ✗（赤）** — 規制値を超えている
- **以内 ✓（緑）** — 規制値の範囲内
- 条文リンク（青字）をクリックすると根拠条文が展開

> **AND条件について**: 法令は複数のパラメータがすべて同時に超過した場合のみ規制対象になる場合があります。  
> 例: 含有量が超過していても比強度が以内なら非該当、という条件があります。

---

### PDF出力

1. 判定結果画面下部の **「🖨️ PDF出力」** ボタンをクリック
2. ブラウザの印刷ダイアログが開く
3. 送信先を **「PDFとして保存」** に選択して保存

PDFには以下が含まれます：
- 判定日時・品目名
- 判定結果と理由
- パラメータ比較表
- 根拠条文（全文）

- 新しい順に一覧表示

## よくある質問

**Q. 品目名を入力してもパラメータが表示されない**  
A. 品目名の表記が法令用語と異なる場合があります。以下を試してください：
- `ガラス繊維` → `無機繊維`
- `CFRP` → `炭素繊維`、`複合材料`
- それでも表示されない場合は「不明・その他」フォームを選択

**Q. 「要人間確認」と表示された**  
A. 入力したパラメータと合致する閾値がDBに見つからない状態です。CISTECへの確認を推奨します。

**Q. AND条件で非該当になったが、本当に問題ないか不安**  
A. このシステムは貨物等省令（リスト規制）を判定対象としています。キャッチオール規制（需要者・用途ベースの規制）は別途確認が必要です。

## 判定結果に疑問がある場合

判定結果に疑問がある場合（対応条文が誤っている、閾値の比較方向がおかしい、製品カテゴリと確認すべき条項が合っていないなど）は、以下の窓口までご連絡ください。開発者が内容を確認し、必要に応じてシステムを修正します。

> 📧 連絡先: `<export-check-support@example.com>`（※導入時に実担当者のアドレスに置き換えること）

報告いただく際は以下の情報を添えてください：
- 品目名・入力したスペック値
- 表示された判定結果
- 期待した結果との差異（例: 「非該当と表示されたが、第四条第十五号が該当するはず」）

## 注意事項

- 本システムは貨物等省令（法令ID: 403M50000400049）に基づいています

- 判定結果は必ずPDF等で保管してください（法的義務）
- 最終判断は必ず輸出管理担当者が確認してください
